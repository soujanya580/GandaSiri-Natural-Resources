🌳 Gandha Siri – Sandalwood Tree Protection & Monitoring System

📌 Problem Statement
Illegal sandalwood tree theft and lack of proper monitoring create major financial and environmental losses for farmers and landowners. Existing systems do not provide an easy way to digitally register, monitor, and protect valuable trees.

Gandha Siri is an Android-based smart tree management application designed to help users register sandalwood trees, capture GPS locations, track growth, manage theft alerts, and access emergency support features.
## 🔗 Live Demo
[Open Gandha Siri Project](https://fanciful-daffodil-68c03c.netlify.app/)

✨ Features

🌱 Tree Registration
- Register sandalwood trees digitally
- Capture tree photo using camera/gallery
- GPS location capture
- Store tree details:
  - Girth
  - Height
  - Year planted
  - Additional notes

📍 GPS Location Tracking
- High accuracy GPS location support
- Tree location monitoring and verification

🚨 Theft Tracking System
- Present/Missing tree verification
- Theft alert monitoring
- Tree security management

🔔 Emergency Security Alert
- SOS panic alert button
- Emergency alert sound system
- Quick emergency support access

🤖 AI Chatbot Support
- Multilingual support
- Kannada / Hindi / English responses
- Text-to-Speech functionality

📊 Maturity Calculator
- Sandalwood maturity estimation
- Growth tracking support

📘 Legal Guide
- Sandalwood ownership information
- Harvesting procedures
- Legal awareness and penalties

📞 Emergency Contacts
- Police emergency support
- Forest department helpline
- Area-specific emergency contacts

▶️ How to Run

1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/GandhaSiri.git
```

2️⃣ Open in Android Studio

- Launch Android Studio
- Open the GandhaSiri project folder

3️⃣ Sync Gradle

- Wait for Gradle sync to complete

4️⃣ Run Application

- Connect emulator or Android device
- Click Run ▶️ button in Android Studio
